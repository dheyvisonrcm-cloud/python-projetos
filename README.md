Este repositório contém meus primeiros testes práticos com a linguagem Python, focados em lógica de programação e tratamento de dados. Os projetos foram desenvolvidos no VS Code e publicados via Git/GitHub. 
Projetos Incluídos:
1. Comparador de Viabilidade (Gasolina vs Álcool)Um script que ajuda o usuário a decidir qual combustível é mais vantajoso financeiramente.O que faz: Calcula a relação de preço entre o álcool e a gasolina (regra dos 70%).Destaque Técnico: Possui validação de dados para garantir que o usuário não digite valores negativos ou zero.
   
2.  Calculadora de Conta de EnergiaUm sistema que simula o cálculo de uma conta de luz com base no consumo e bandeiras tarifárias.O que faz: Multiplica o consumo (kWh) pela taxa base e adiciona acréscimos conforme as bandeiras Verde, Amarela ou Vermelha.Destaque Técnico: Utiliza blocos try/except para impedir que o programa trave caso o usuário digite letras em vez de números.
   
3.   Tecnologias e Conceitos Utilizados:
     linguagem: Python 3.x
     Entrada de Dados: input() com conversão para float.
     Estruturas de Controle: if, elif, else.
     Loops de Repetição: while True e continue para persistência de dados.
     Tratamento de Exceções: Uso de ValueError para blindagem do código.
